# STX Staking Frontend

React frontend for the STX staking contract using @stacks/connect and @stacks/transactions.

## Features

- Connect Hiro Wallet
- Stake STX tokens
- Unstake STX tokens
- View staked balance

## Installation

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Dependencies

- React 18
- @stacks/connect
- @stacks/transactions
- Vite

## Usage

1. Connect your Hiro Wallet
2. Enter amount in STX
3. Click Stake or Unstake
4. Confirm transaction in wallet

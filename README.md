# Staking Project

STX staking contract for Stacks blockchain

## Features
- Stake STX tokens
- Unstake STX tokens
- Track total staked amount

## Installation
\`\`\`bash
clarinet check
\`\`\`

## Usage
Call the stake function with amount in micro-STX

## Testing
Run tests with npm test

## Requirements
- Clarinet 2.0+
